Tutorial de uso

Conoce el flujo de trabajo del sistema: desde que el mesero toma una orden hasta que se cierra la cuenta.

person
Mesero
Toma la orden en la mesa y la envía a cocina
countertops
Cocina
Recibe las comandas y prepara los platos
payments
Cajero
Genera la factura y cobra al cliente
settings
Administrador
Configura el sistema, usuarios e inventario
1 Flujo de atención al cliente

Así funciona el ciclo completo desde que el cliente llega hasta que paga:

table_bar
1. Ocupar mesa
El mesero asigna una mesa al cliente
arrow_forward
edit_note
2. Tomar orden
Registra los productos en una comanda
arrow_forward
countertops
3. Cocina
Cocina recibe y prepara
arrow_forward
payments
4. Facturar
Cajero genera la cuenta y cobra
lightbulb

Importante: El mesero no imprime el voucher. La factura la genera el cajero en la PC de facturación, y el mesero la retira para entregarla al cliente.

2 Gestión de mesas
table_restaurant
Plano del salón
Vista gráfica de todas las mesas con su estado: libre, ocupada o facturando.
touch_app
Acciones rápidas
Toca una mesa para abrir el menú: ocupar, crear comanda, ver cuenta, facturar o liberar.
3 Comandas (pedidos)
add_circle
Crear comanda
Selecciona productos del menú, ajusta cantidades y agrega notas. La comanda se envía a cocina automáticamente.
receipt_long
Lista de comandas
Historial de pedidos con filtros por estado: abiertas, en cocina, cerradas.
info

Nota: Una comanda puede tener varios estados. Cuando el mesero la crea, pasa a En cocina. Al prepararse, cocina la marca como Lista. Al facturarse, se cierra automáticamente.

4 Vista de cocina
countertops
Pantalla de cocina
Muestra en tiempo real las comandas pendientes. Cada pedido aparece con el número de mesa, productos y notas.
check_circle
Marcar como listo
El cocinero marca la comanda como lista cuando termina de preparar el plato.
5 Facturación y cobro
description
Generar factura
Selecciona los productos consumidos, asigna mesero y el sistema calcula subtotal, IVA (15%), servicio (10%) y total.
payments
Registrar pago
Efectivo, tarjeta o ambos. El sistema calcula el cambio y registra el pago. Se imprime el voucher desde la PC de caja.
savings
Cierre de caja
Al final del turno, el cajero genera un reporte con total de ventas, formas de pago y diferencias.
takeout_dining
Para llevar
Crea facturas sin mesa asignada para pedidos de carry-out. Mismo proceso de cobro.
print

Impresión: Los vouchers se imprimen desde la PC de facturación donde corre el agente de impresión. Revisa la sección Ayuda Impresión para configurarlo.

6 Inventario y compras
pallet
Control de stock
Registra ingredientes y recetas. El sistema descuenta automáticamente del inventario al facturar.
shopping_cart
Compras
Registra compras de ingredientes a proveedores, con seguimiento de cuentas por pagar.
swap_horiz
Movimientos
Ajustes manuales de inventario: entradas, salidas, mermas y transferencias.
business
Proveedores
Directorio de proveedores con datos de contacto e historial de compras.
7 Administración del sistema
group
Usuarios
Crea y gestiona usuarios del restaurante. Asigna roles (Mesero, Cajero, Cocinero, Admin) con permisos específicos.
admin_panel_settings
Permisos
Configura qué puede ver y hacer cada rol: acceso a dashboard, mesas, comandas, cocina, facturación e inventario.
settings
Configuración
Datos del restaurante, nombre, dirección, RUC, teléfono. También incluye la zona de peligro para restaurar respaldos o restablecer de fábrica.
cloud_backup
Respaldos
Crea copias de seguridad completas o por restaurante. Descarga, restaura o elimina respaldos desde la sección dedicada.
lightbulb Consejos rápidos
  • Usa el menú lateral (icono menu) para navegar entre secciones.
  • Los meseros pueden usar el sistema desde un celular o tablet sin problemas.
  • La cocina funciona mejor en una pantalla grande o tablet fija en la cocina.
  • Los vouchers se imprimen únicamente desde la PC de facturación con el agente de impresión activo.
  • Activa el modo oscuro desde el icono dark_mode en la barra superior.
  • Revisa la sección Ayuda Impresión si tienes problemas para imprimir.
