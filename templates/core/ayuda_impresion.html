Configurar Impresora Térmica

Sigue estos pasos para imprimir tickets y comandas directamente en tu impresora térmica.

1 Requisitos
  • Windows 7, 10 u 11
  • Python 3.8 o superior instalado
  • Impresora térmica conectada por USB o red
2 Instalar dependencia

Abre una terminal (CMD o PowerShell) como administrador y ejecuta:

pip install pywin32
3 Iniciar el agente de impresión

Opción A — Con doble clic (para probar):

Navega a la carpeta del sistema y haz doble clic en printer_agent.py.

Opción B — Manual desde terminal:

cd C:\ruta\del\sistema
python printer_agent.py

Opción C — Instalar inicio automático Recomendado

Haz doble clic en install_agent.bat (una sola vez). A partir de ahora el agente se iniciará solo al encender la computadora.

4 Verificar que funciona

Cuando el agente esté corriendo verás en la terminal:

MKSOFT Printer Agent corriendo en http://127.0.0.1:8765
Impresora detectada: EPSON TM-T20II

Si no detecta la impresora automáticamente, verifica que esté encendida y conectada. El agente busca impresoras con nombres que contengan: TM, Thermal, POS, Ticket, Receipt, Epson, Star, Bixolon, Zebra.

5 Probar impresión

Abre cualquier factura o comanda y haz clic en el botón Imprimir de la barra de herramientas. El sistema enviará el ticket directamente a la impresora térmica.

6 Solución de problemas
Problema Solución
“Agente no disponible”Ejecuta printer_agent.py o install_agent.bat
“pywin32 no instalado”Ejecuta pip install pywin32
“No se detectó impresora térmica”Verifica conexión USB / nombre de la impresora
La impresión sale en blancoRevisa que el papel térmico esté bien colocado
Caracteres extrañosLa impresora no soporta ESC/POS. Usa impresora térmica compatible
Error “Permission denied”Ejecuta la terminal como administrador
Puerto 8765 ocupadoCierra otros programas o usa python printer_agent.py --port 8766
lan Impresión por red (TCP/IP)

Si tu impresora está en red con IP fija, puedes imprimir directamente añadiendo ?host=192.168.1.100 a la URL de impresión. El puerto por defecto es 9100.

Ejemplo: {% url 'facturacion:escpos' 1 %}?host=192.168.1.100
